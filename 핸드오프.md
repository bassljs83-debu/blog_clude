> ⚠️ **연속성은 이제 git(비공개 GitHub)으로 관리한다.** 구글 드라이브는 폐기.

| 단계 | 내용 | 호스트 |

### 콘텐츠 전략 — ⭐ 2트랙 분리 (프로젝트 생존의 핵심)
2026 구글: **진짜 경험 +22% / AI 짜깁기 -71%**, 대량 AI생성은 스팸정책(scaled content abuse) 위반 → 사이트째 제재. 그래서:
- **정보/비교/구매가이드형** (대량생산 주력) → 경험 주장 안 함. 스펙·가격·리뷰종합·비교표 기반. "이런 기준으로 골랐다"
- **경험 후기형** (소수 정예) → 실제 구매 상품에만. 진짜 사진·진짜 디테일. 클러스터의 신뢰 앵커

## SEO 원칙 (2026 구글 로직 연구 결과)
1. **Experience가 최강 신호** — 우리 방향이 맞음. 단 AI 티 나면 -71% 표적 → 문체검증이 SEO 방어선
2. **AI Overviews가 리뷰/제휴 클릭 28~45% 잠식** → 빅키워드 X, **롱테일+경험+인용가능 구조**로
3. **Helpful Content는 사이트 단위** → 블로그 전체 people-first. 소개/개인정보/문의 페이지 필수. 토픽 클러스터
4. **구조화·인텐트 일치**: 롱테일 제목 · 두괄식 TL;DR · 질문형 H2(PAA) · 스펙 표 · 하단 FAQ
5. 애드센스/기술: 글당 800자+, 이미지 1~2장, Core Web Vitals

## 생성기 필수 골격 (템플릿 강제 슬롯)
1. 롱테일 제목 `[상황]+[대상]+[상품]+[후기/방법]`
2. 두괄식 3줄 요약 (featured snippet + AI 인용)
3. 질문형 H2 (People Also Ask 대응, 각 H2 첫 문장은 직답)
4. 스펙 비교 표 (기계 추출용)
5. 하단 FAQ
6. 경험/대가성 문구 (대가성 문구 필수)

### 검증 루프 (차별점)

- ✅ `coupang.py` — 쿠팡 API 모듈 완성 & **라이브 테스트 통과** (제습기 로켓 9개 수집)
- ✅ `.venv` 실행환경 (`python3 -m venv .venv`, requirements 설치됨)
- ✅ `.env` 에 ANTHROPIC_API_KEY 입력 완료 (Claude API)
- ✅ `generator.py` — Claude API 글 생성기 완성 & **라이브 테스트 통과**
  - 제습기 정보/비교형 글 9,341자, 6종 골격 전부 + 파트너스 링크 9개 정상

### generator.py 실전 메모
- **실행법**: `.venv/bin/python generator.py "제습기"`
- 흐름: coupang 상품수집 → 초안(1차 호출) → 검증·교정(2차 호출) → `output/` 저장
- 모델: 기본 `claude-opus-4-8`. `.env` 의 `GENERATOR_MODEL` 로 변경 가능
  (Phase 2 대량생산 시 비용 낮추려면 `claude-sonnet-5` 등으로)
- 정보/비교형 트랙만 구현됨. 경험 후기형 트랙은 추후

### coupang.py 실전 메모
- **실행법**: `.venv/bin/python coupang.py "제습기"`
- **search limit 최대 10** (11부터 400 `limit is out of range`) → `SEARCH_MAX_LIMIT` 로 클램프
- **rCode 0 = 성공.** rMessage 는 성공 시에도 "대가성 문구 넣으세요" 안내가 옴 (에러 아님)
- **검색 결과 productUrl 에 이미 파트너스 수익 링크(lptag) 포함** → deeplink 변환 없이 바로 사용 가능
- 파이썬 3.9 라 파일 상단 `from __future__ import annotations` 필요

## 실행 방법 (운영자용) ⭐
- **`블로그생성.command` 더블클릭** → 창에서 주제 입력 → 글+태그+썸네일 자동 생성 → 결과 폴더 열림
  - 주제 직접 입력(예: 제습기) 또는 **엔터 = 자동 주제 선택**(`categories.txt` 목록에서)
  - `q` = 종료. 터미널 지식 불필요.
- 자동 주제 목록은 `categories.txt` (메모장으로 편집 가능)
- 결과물: `output/` 에 `.html`(붙여넣기용) · `_태그.txt` · `_thumb.png` · `.md` · `_products.json`
